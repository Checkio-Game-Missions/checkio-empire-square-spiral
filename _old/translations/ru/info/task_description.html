Никола подобрал странную монтажную плату. Все элементы на данной плате упорядочены по спирали и можно соединить соседние элементы по вертикали или горизонтали.

Схема цепи представлена в виде квадратных ячеек. Первый элемент находится в центре и помечен как 1 и далее по спирали по часовой стрелке, каждый элемент помечен в возрастающем порядке. На схеме, вы можете соединять ячейки по вертикали и горизонтали. Вы можете помочь Николе найти манхэттенское расстояние между двумя любыми элементами на схеме. Для примера, расстояние между ячейками 1 и 9 -- два и расстояние между 24 и 9 -- один.

Вх. данные: Две метки ячеек, как целочисленные (int).

Вых. данные: Манхэттенское расстояние между двумя указанными ячейками, как целочисленное (int).

Примеры: 

find_distance(1, 9) == 2
find_distance(9, 1) == 2
find_distance(10, 25) == 1
find_distance(5, 9) == 4

Как это используется: В этой задаче вы можете познакомится с нестандартными системами координат. Иногда данная координатная модель используется в топографии ("улитка"). Также эта задача связана с "Скатерью Улама" Скатерть Улама — названная в честь Станислава Улама спираль чисел натурального ряда, на которой отмечены клетки, соответствующие простым числам.

Предусловия: 0 < first < 1000
0 < second < 1000
first != second
